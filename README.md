# 🎓 SkillBridge Frontend

SkillBridge is a premium 1-on-1 tutoring marketplace that connects passionate learners with expert mentors. This repository contains the **Next.js** frontend, featuring a stunning, responsive UI designed for a seamless learning experience.

## ✨ Key Features

### 👤 For Students
- **Browse & Search:** Find tutors by subject, rating, and price.
- **Instant Booking:** Schedule sessions with a real-time availability check.
- **Learning Dashboard:** Track upcoming sessions and view completed lesson history.
- **Reviews & Ratings:** Leave feedback for tutors after completed sessions.
- **Profile Management:** Update personal details, bio, and location.

### 👨‍🏫 For Tutors
- **Profile Customization:** Set hourly rates, experience, and expert categories.
- **Availability Management:** Define weekly time slots for student bookings.
- **Tutor Dashboard:** Monitor upcoming teaching sessions and student reviews.

### 🛡️ Admin Panel
- **User Management:** Oversee all students and tutors, with the ability to manage account status.
- **Platform Analytics:** View total bookings, active users, and category distributions.

## 🚀 Getting Started

### 1. Prerequisites
- Node.js (v18+)
- npm or yarn

### 2. Installation
```bash
git clone https://github.com/rubel6610/skillbridge-frontend.git
cd skillbridge-frontend
npm install
```

### 3. Environment Setup
Create a `.env` file in the root directory:
```env
NEXT_PUBLIC_BASE_URL=http://localhost:5000/api
```

### 4. Run Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to see the application.

## 📸 Design Philosophy
SkillBridge uses a **Premium Light Theme** characterized by:
- **Luminous Layouts:** High-contrast Slate-900 typography on clean White/Indigo-50 backgrounds.
- **Glassmorphism:** Subtle borders and soft indigo shadows for a modern, "floating" UI feel.
- **Modular Components:** Refactored sections for maximum maintainability and performance.
